# AutoGLM-Tactical-GUI
这是一个基于 Python Tkinter 开发的 AI 手机自动化控制台。它主要通过图形化界面驱动 AutoGLM 模型，实现对 iOS/Android 设备的智能操控。  

核心功能：  战术化 UI：采用赛博朋克风格终端，实时分屏显示 AI 的“思考过程”与“执行动作”。  

三大作战模式：支持单次打击（即时任务）、延时执行（定时启动）及递归监控（自动循环轮询，如每隔 5 分钟检查一次微信）。  

自动化链路：集成 iOS 的 WDA 诊断与 iproxy 转发逻辑，支持自定义 API 密钥与 Prompt 指令。  

安全熔断：具备多线程处理与紧急停止功能，确保自动化过程可控。  

一句话总结： 这是一个为 AutoGLM 打造的、具有强视觉冲击力的“智能体指挥部”。
